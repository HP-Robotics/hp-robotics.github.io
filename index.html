Rules of HP Robotics Programming

  1. All team laptops use the team GitHub user. Ask a teammate or a mentor if you are unsure which user this is.
  2. Personal computers use your own GitHub accounts.
  3. Helpful links
    • Chief Delphi: www.chiefdelphi.com
    • WPI Lib: Docs.wpilib.org
  4. All robot code is located under "Automatons Code" from the shortcut on the desktop or in C:/Users//Automatons Code
  5. To get new code from GitHub, do the following:
    • Go to our github.com page and click on the green clone icon and copy the git https URL
    • In Visual Studio Code, do File..Open and open the directory that corresponds to Automatons Code.
    • Select Terminal..New
    • Type 
      git clone
      and then paste the git URL and hit enter
